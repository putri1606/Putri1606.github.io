Tentang Saya

Saya Putri Mawar Fadillah biasa dipanggil Putri, Mawar, Fadillah, Putmaw atau Lilah. Saya lahir di Wonosari, Kecamatan Kinali, Provinsi Sumatra Barat pada tanggal 16 Juni 2025. Saat ini saya berusia genap 20 tahun. Saya tinggal di Jakarta, tepatnya di Kecamatan Tebet, Kota Jakarta Selatan. Saya memiliki banyak hobi, yaitu memasak, membuat aneka kue, menonton film, berpergian, dan bermain sepeda. Saya paling sering memasak dan membuat kue, karena saya membuka bisnis kecil dari rumah berupa bisnis makanan, seperti aneka kue, bolu, kukis, aneka olahan pisang, dan aneka gorengan.

Saya memulai pendidikan dasar saya di Padang, yaitu SDN 12 Kinali dan SMPN 02 Kinali. Pada tahun 2020 tepat saat indonesia mulai terjangkit virus corona. Saat itu saya pindah ke Jakarta dan mulai besekolah di Jakarta. Saya memulai di SMPN 265 Jakarta dan melanjutkan SMA di SMAN 8 Jakarta. Pada tahun 2024, saya lulus SMA dan melanjutkan ke jenjang perguruan tinggi. Namun, semua itu belum rezeki saya, sehingga saya harus melalui masa tahun jeda selama satu tahun. Saat itu, saya mengisi hari-hari kosong saya dengan magang di sebuah yayasan selama 3 bulan untuk menambah pengalaman dan ilmu baru. Setelah memasuki semester genap, saya mulai les kembali seperti dahulu lagi untuk persiapan ujian perguruan tinggi negeri. Saya juga les di Alhazen Academy untuk memulai lebih dulu.
